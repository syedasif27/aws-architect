# 30-Day AWS Solutions Architect Training Plan

## Week 1: Foundations (AWS Basics & Core Services)
**Goal:** Understand AWS fundamentals, key services, and the exam structure.  
- **Day 1:** Create AWS Free Tier account, learn Global Infrastructure, read Well-Architected Framework.  
- **Day 2:** IAM – Users, Groups, Roles, MFA. *Lab:* Create IAM users and policies.  
- **Day 3:** EC2 – Instances, AMIs, Security Groups. *Lab:* Launch EC2, install Apache.  
- **Day 4:** EBS, EFS, Instance Store. *Lab:* Add/resize EBS.  
- **Day 5:** S3 – Buckets, ACLs, Lifecycle. *Lab:* Create bucket, enable versioning.  
- **Day 6:** VPC Basics – Subnets, IGW, NAT, SGs vs NACLs. *Lab:* Create VPC with public/private subnet.  
- **Day 7:** Quiz + Revision.  

## Week 2: Advanced Networking & Databases
**Goal:** Learn networking deeply and start DB services.  
- **Day 8:** ELB + Auto Scaling. *Lab:* Deploy ALB + ASG with EC2.  
- **Day 9:** RDS – Engines, Multi-AZ, Read Replica. *Lab:* Launch RDS MySQL.  
- **Day 10:** DynamoDB basics.  
- **Day 11:** Route 53 – Hosted Zones, Routing. *Lab:* Map domain to EC2.  
- **Day 12:** CloudFront & S3 static hosting. *Lab:* Deploy static site.  
- **Day 13:** VPC Advanced – Peering, Endpoints, Transit Gateway.  
- **Day 14:** Quiz + Practice questions.  

## Week 3: Security, Serverless, Monitoring
**Goal:** Learn key exam-relevant services.  
- **Day 15:** KMS – Encryption basics.  
- **Day 16:** CloudWatch (metrics, alarms) & CloudTrail.  
- **Day 17:** SNS, SQS basics.  
- **Day 18:** Lambda. *Lab:* S3 trigger Lambda.  
- **Day 19:** Elastic Beanstalk basics.  
- **Day 20:** AWS Organizations & Cost Explorer.  
- **Day 21:** Quiz + Case studies.  

## Week 4: Architecture + Exam Prep
**Goal:** Focus on exam readiness and architecture design.  
- **Day 22:** High Availability & Disaster Recovery.  
- **Day 23:** Migration & Hybrid Services – Storage Gateway, Snowball, DMS.  
- **Day 24:** Exam Case Studies – Multi-tier, HA web apps.  
- **Day 25:** Well-Architected Framework deep dive.  
- **Day 26:** Practice Exam (Set 1) – Review mistakes.  
- **Day 27:** Hands-on labs (revisit weak topics).  
- **Day 28:** Practice Exam (Set 2).  
- **Day 29:** Whitepapers (exam guide, Well-Architected, Security).  
- **Day 30:** Final Mock Exam – Timed full test.
